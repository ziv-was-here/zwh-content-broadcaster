# WordPress.org Plugin Assets

To make your plugin look professional on WordPress.org, you should add the following images to this folder (and eventually to the `assets/` directory in SVN):

1. **icon-128x128.png** (or .jpg) - Small icon
2. **icon-256x256.png** (or .jpg) - Large icon (High-DPI)
3. **banner-772x250.png** (or .jpg) - Normal banner
4. **banner-1544x500.png** (or .jpg) - High-DPI banner

These images will appear on your plugin's page in the WordPress.org directory and in the WP admin "Add New Plugin" search results.
